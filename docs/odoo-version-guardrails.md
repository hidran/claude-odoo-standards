# Guardrails de versión Odoo (v14 → v19)

> **Movido.** La matriz de guardrails ahora es la **fuente única** en la skill
> `plugins/odoo-standards/skills/odoo-version-guardrails/SKILL.md`, que Claude carga
> automáticamente al trabajar vistas/modelos. Se mantiene un solo sitio para evitar
> divergencias.
>
> Para leerla como humano, abre ese `SKILL.md`. Para aplicarla, el plugin lo hace por ti
> según el `🔒 VERSION LOCK` del repo de cliente.
